## Автор

Парамзина Екатерина Андреевна

## Функционал

- Загрузка списка сотрудников с API
- Поиск по имени, фамилии, никнейму (с debounce)
- Фильтрация по отделам
- Сортировка (по алфавиту / по дню рождения)
- Группировка по годам (для сортировки по дню рождения)
- Страница деталей сотрудника
- Адаптивная вёрстка
- Обработка ошибок и состояний загрузки
- Отслеживание состояния сети, оранжевый баннер при отсутствии интернета, данные из кэша
- Фиолетовый баннер при восстановлении соединения, автоматическое обновление данных
- Кэширование результатов
- Сохранение состояния фильтрации
- Тёмная тема
- Локализация 


## Стек технологий

- React + TypeScript
- React Router
- Axios
- Styled Components
- Vite
- i18next
- LocalStorage
